maybe a good night

最后更新: 10月 21, 2025 作者: LDN97 字数: 558 预计阅读时间: 1min

現在大概是凌晨一點，也差不多把這個網站架好了。想著網站上線前，總得至少有一篇文章，所以打算隨手紀錄一下。

其實架網站沒有想像中那麼難，有 Hexo 這種東西加上別人的主題模板，熟練一點大概半天就能搞定。
不過第一次寫文章反而有點不知道該寫什麼，平常躺在床上發限動明明隨便寫就一大堆的。

會做這個網站的動機其實很簡單，就是看到群友發了自己的 blog，又看到美術生朋友也有自己的小網頁，想著「別人都有，我怎麼能沒有」，就動手做了這個網站。

做這個網站最大的困難，除了不知道文章要寫什麼之外，大概就是處理背景圖片了。
這個網站主題是來自 「illlights」 的 Klise-glassy ，當時是在研究吉里吉里引擎解包的時候看到他的文章。第一眼就被他的網站風格吸引，後來在找主題時也是第一個想到他的風格 —— 真的很好看啊。

扯遠了。
為什麼說處理背景圖麻煩呢？因為這個作者原本是用自己的圖片 API 來實現隨機背景的功能（而且背景好多都是 galgame，理工男怎麼都這樣啊）。
最後我還是把他的 API 拿掉，改成放上自己喜歡的圖片。結果花了一整晚到處找圖，最後還是找了一堆 gal 的 CG（理工男怎麼都這樣啊）。

也沒什麼好說的，就先這樣吧。
接下來應該處理一下友鏈的部分，就能準備上線了。
我還沒研究怎麼部署網站，或是有哪些需要注意的地方，不過就一個靜態網站應該也沒什麼 Token 問題吧。
我比較擔心的是更新和流量的問題，不過晚點再說吧。

我在想，今晚把網站上線之後要不要丟給我媽看，然後明天跟她說我整個晚上都在搞這個，讓我請個假寫點東西。哈哈，大概是不行吧。
